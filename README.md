## About This Project

This is a professional landing page for an independent **Artificial Breeding (AI) Technician** based in **Ngakuru, New Zealand**, built to serve the dairy farming industry during the 2026 mating season.

The site allows farmers to view service information and book a mating date and time slot online \u2014 designed for simplicity and ease of use, including on mobile devices in the field.

        ## Features

        - **Online Booking Form** \u2014 Farmers can submit their details and preferred dates easily
        - - **Interactive Service Map** \u2014 Built with Leaflet.js, centred on Ngakuru with a 50 km radius overlay
          - - **Live ETA Tracking** \u2014 Farmers can track the technician's ETA
            - - **Mobile Optimised** \u2014 Fully responsive layout for on-farm use
              - - **QR Code Access** \u2014 Easy on-site scanning for quick access to the booking page
                - - **Professional Branding** \u2014 Clean, high-contrast pink & white design suited for outdoor screen use
